Model calibration decisions
MODEL SELECTION
DeBERTa (33M params) via Ollama
Chosen over BERT for memory efficiency on local device. Validated on self-written test phrase before full-run: 90–96% confidence, correct classification.
THRESHOLD CALIBRATION
0.90 → 0.80 → 0.70 → 0.75 ✓
0.90 forced single-category only. Iterated to find balance between precision and dual-category coverage. 0.75 was the middle ground.
TOP-K CONSTRAINT
Top-K = 2 per review
Each review assigned to at most 2 categories above threshold. Prevents over-labelling while capturing multi-theme complaints.
